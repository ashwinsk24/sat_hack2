- Python 3.9
- Pynecone
- NodeJS 16.8
- SQLite3

## How to configure
- Installing
```
$ pip install pynecone
```
- Create a Project
```
$ mkdir my_app_name
$ cd my_app_name
$ pc init
```
## How to Run
```
$ pc run
```
